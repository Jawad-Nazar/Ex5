# Multi Counter App

A simple Android app that displays three independent counters, each with its own controls. Users can increment, decrement, and set custom starting values.

# Features

Three independent counters (stacked vertically)

Each counter includes:

“+” button to increase the count

“–” button to decrease the count

Text field to enter a custom starting value

Clean row layout for each counter

Fully state-independent counters

# Tech Stack

Android (Kotlin)

Jetpack Compose (recommended) or XML Layouts
